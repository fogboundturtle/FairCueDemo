# FairCue Demo

## Demo Features
- Manage a player database
- Create and run chip tournaments
- Create and run bracket tournaments
- Generate rounds and matchups
- Show round announcement windows
- Manage purse structures
- View tournament history
- Use English and French UI

## Demo Limits
- 14-day trial period
- Maximum 16 tournament players
- Not available in demo:
  - Theme Editor
  - PDF export
  - Player import/export
  - Custom PDF logo
  - Single and Team League Support
  - Round Robin Tournament
  - Stream overlay and Stream companion App
  - Challonge integration for Single/Double elimination Bracket
